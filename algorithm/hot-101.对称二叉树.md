# 对称二叉树
## 问题
```
给你一个二叉树的根节点 root ， 检查它是否轴对称。
示例 1：
输入：root = [1,2,2,3,4,4,3]
输出：true
示例 2：
输入：root = [1,2,2,null,3,null,3]
输出：false
```

## 答案
```
var isSymmetric = function(root) {
    function dfs(p,q){
        if(p==null||q==null){
            return p===q;
        }
        return (p.val===q.val)&&dfs(p.left,q.right)&&dfs(p.right,q.left);
    }
    return dfs(root.left,root.right);
};
```
