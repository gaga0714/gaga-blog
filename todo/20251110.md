# uv python包管理器
- uv python dir 查看python安装目录
- uv python list 查看已安装python版本
- uv python install xx 安装xx版本

# uv创建虚拟环境步骤
进入项目
```bash
cd uapp
```
创建虚拟环境，命名xxx
```bash
uv venv xxx --python 3.8 
```
激活环境
```bash
xxx\Scripts\activate
```
安装依赖
```bash
uv sync
```
