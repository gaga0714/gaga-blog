# es6数组的扩展
...把数组转成逗号分隔的参数序列，浅拷贝，解构赋值，只能放最后一位
[..."hello"]->["h","e","l","l","o"]
可迭代对象才能使用...

Array.from()把类似数组的对象和可遍历对象变成数组
Array.from(arr,(x)=>x*x);

Array.of()没有参数返回空数组，一个参数指定长度等于几个逗号数，2个及以上返回新数组

copyWithin(开始替换的下标，开始拿数据的下标，拿到哪结束的下标)
find找第一个符合的数值，findIndex找第一个符合的下标
都可以接收第二个参数指定this绑定

fill(数，开始，结束)
entries,keys,values
includes(值，起始)
flat(n)拉平n层，默认不填是一层   flatMap()

# 遍历对象
for in 不含不可枚举，不含symbol
Object.keys 不含继承，不含symbol，不含不可枚举，
Object.getOwnPropertyNames 不含继承，不含symbol
Reflect.ownKeys(obj) 不含继承的所有

遍历顺序：
先数值键，按数值升序
字符串键，按加入时间升序
Symbol键，按加入时间升序

其他和===一致
Object.is(NaN,NaN)//true
Object.is(+0,-0)//false

Object.assign(target,src1,src2)同名属性会替换
Object.fromEntries()将键值对数组转为对象

# Promise

三种状态：
pending进行中
fulfilled成功
rejected失败

Promise构造函数接收一个**函数**作为参数
```js
const promise=new Promise(function(resolve,reject){

})
```
实例方法：then，catch，finally

构造函数方法：all race allSettled resolve reject try
