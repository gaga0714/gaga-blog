# Network

其实前端关心的就是浏览器和服务器之间的对话：从输入 URL 开始，DNS、TCP、TLS、HTTP，再到浏览器拿到字节流后的解析、渲染、缓存。

HTTP（1.0/1.1/2/3 的演进）、TCP/UDP、TLS、跨域、缓存、安全～
并非孤立知识点，而是一条完整链路——任何一环慢，页面就慢；任何一环漏，页面就不安全。
