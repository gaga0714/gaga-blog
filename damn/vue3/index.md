# Vue3

## 再次开堂

渐进式 JavaScript 框架。

Vue3 相比 Vue2 最大的变化：Composition API、Proxy 代替 Object.defineProperty、性能提升（PatchFlag、Block Tree）、更好的 TS 支持。
