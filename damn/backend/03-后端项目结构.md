- 微服务架构
- SOA 架构
- 多模块

```
my-app-parent
├─ pom.xml
├─ my-app-api        # 对外接口定义：DTO、VO、Feign接口（可选）
├─ my-app-common     # 通用：工具、异常、统一返回、基础配置
├─ my-app-domain     # 领域模型：Entity/Domain对象、领域服务（可选）
├─ my-app-infra      # 基础设施：DB、MQ、缓存、第三方SDK、Repository实现
└─ my-app-web        # Web入口：Controller、启动类、Web配置

```
