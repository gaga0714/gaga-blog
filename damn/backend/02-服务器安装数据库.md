# 服务器安装数据库
阿里云服务器，安装docker后使用docker安装mysql
## 下载mysql
直接下载最新版本
```bash
docker pull mysql
```
若要下载指定版本，去链接上找：
https://hub.docker.com/_/mysql/tags

## 查看镜像
```bash
docker images
```
## 运行镜像
创建data目录：`sudo mkdir /data/mysql`

运行容器：`docker run -itd --name mysql-lamp -p 3306:3306 -e MYSQL ROOT PASSWORD=123456 -v /data/mysql:/var/lib/mysql/ mysql`

## 查看运行的容器
`docker ps` 当前运行的

`docker ps -a` 所有状态的

查看host，修改host
```bash
docker exec -it mysql-lamp mysql -uroot -P3306 -hlocalhost -p123456
```
