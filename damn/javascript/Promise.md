# Promise

Promise 是 JavaScript 用来处理异步操作的一种对象。它表示一个未来可能完成或失败的操作。

**一个 Promise 代表了一个还没完成但最终会完成或失败的任务。**

它有三种状态,一旦`fulfilled`或`rejected`，状态就不可再更改

- pending（进行中）

- fulfilled（已成功）

- rejected（已失败）
