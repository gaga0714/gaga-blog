# V8 编译

首先对源代码进行词法分析，生成tokens；然后对tokens进行语法分析，生成AST抽象语法树；编译器将AST转换成字节码；解释器对字节码进行解释执行；

如果发现有热点代码，JIT编译器将字节码转成机器码，直接在cpu上跑，更高效；

创建上下文是在代码执行阶段，而不是编译阶段
