## IntersectionObserver
```js
const options={
    threshold:1.0,
    root:document.querySelector('#scrollArea')
};
let observer=new IntersectionObserver();
observer(callback,options);
//传入被观察者
const target=document.querySelector('.target');
observer.observe(target);
```
