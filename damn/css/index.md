# CSS - 样式

## 依旧碎碎念

CSS是什么？ 标准回答：Cascading Style Sheets，层叠样式表。

其实就是**样式**，生成一棵 CSSOM 树，和DOM树一起构建渲染树。

三个核心：层叠（Cascade）、特异性（Specificity）、继承（Inheritance）。

如何摆放盒子？ -> 布局演进史：table → float → flex → grid
