2.  
```js
let map=new Map(); 
map.set(true,0); 
map[true]=10; 
console.log(map.get(true)); 
console.log(map.get('true')); 
console.log(map['true']);
```
map[true] = 10 是给 Map 实例对象本身挂了一个普通属性。方括号属性访问会把非字符串键转成字符串，所以等价于 map['true'] = 10，这不会改动 Map 的键值对。

Map 的键不做类型转换，true（布尔）和 'true'（字符串）是两个不同的键。因此 map.get('true') 查的是 Map 内部键 'true'，并不存在 → undefined
