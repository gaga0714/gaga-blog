## 快速开始
看别人都是fork了仓库项目，嗯嗯我也这样干了，然后就没有然后了...

所以为什么呢？

别管了，按官网教程造一造先

安装： `pnpm add -D vitepress`

创建项目： `pnpm vitepress init`

```

│
```
