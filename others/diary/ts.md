TypeScript 官方提供的编译器叫做 tsc，可以将 TypeScript 脚本编译成 JavaScript 脚本。本机想要编译 TypeScript 代码，必须安装 tsc。

根据约定，TypeScript 脚本文件使用.ts后缀名，JavaScript 脚本文件使用.js后缀名。tsc 的作用就是把.ts脚本转变成.js脚本。

## 全局安装ts
`npm install -g typescript`

查看版本：

`tsc -v`

显示基本可用选项：
`tsc -h`

完整帮助信息：
`tsc --all`
